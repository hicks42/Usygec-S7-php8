Ce lien ouvre directement le formulaire d'avis Google pour votre établissement

Comment utiliser ce lien ?

  1. Cliquez sur le bouton "Copier" pour copier le lien dans votre presse-papiers
  2. Partagez ce lien avec vos clients (email, SMS, QR code, etc.)
  3. Quand ils cliquent dessus, le popup Google "Donner un avis" s'ouvre directement
  4. Le bouton "Tester" vous permet de vérifier que le lien fonctionne

Note technique :

Ce fichier HTML est juste un exemple visuel. Dans votre application Symfony, le lien sera généré automatiquement à partir du Place ID enregistré dans la base de données.
